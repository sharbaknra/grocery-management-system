# User Guide

## Roles

### Store Manager (Admin/Manager)
**Access**: Full system access

**Features**:
- View dashboard with KPIs
- Manage products (create, edit, delete)
- Manage stock levels
- Manage suppliers
- View orders and invoices
- Generate reports
- Manage staff accounts

### Staff (Cashier)
**Access**: POS and order management

**Features**:
- Point of Sale (POS) interface
- Process sales
- View order history
- View invoices
- Cannot access financial summaries

### Purchasing Agent
**Access**: Purchasing and supplier management

**Features**:
- View purchasing dashboard
- View suppliers and reorder sheets
- View stock levels (read-only)
- Cannot create products or update stock

### Customer
**Access**: Product browsing and ordering

**Features**:
- Browse products
- Add to cart
- Checkout
- View own order history

## Common Workflows

### Making a Sale (Staff)
1. Login to POS
2. Search/add products to cart
3. Review cart
4. Complete checkout
5. Order created and stock deducted

### Managing Inventory (Manager)
1. Navigate to Stock Levels
2. View current stock
3. Click "Update quantity" to restock
4. Enter quantity and reason
5. Stock updated and logged

### Generating Reports (Manager)
1. Navigate to Reports
2. Select report type
3. View report data
4. Export as CSV/PDF if needed
